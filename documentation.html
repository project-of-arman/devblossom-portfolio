Download & Installation

1. Download from ThemeForest

After purchasing DevBlossom Portfolio, you can download the package from:

  1. Go to your ThemeForest Downloads page
  2. Find DevBlossom Portfolio in your purchases
  3. Click "Download" and select "All Files & Documentation"

2. Extract Files

The downloaded package includes:

  • devblossom-portfolio.zip - Main template files
  • documentation.html - This documentation
  • changelog.md - Version history

3. Install Dependencies

Open your terminal and run:

# Navigate to template directory
cd devblossom-portfolio

# Install dependencies
npm install

# Start development server
npm run dev

Customization

Theme Customization

The template uses Tailwind CSS for styling. You can customize colors, typography, spacing, and more in: 

// tailwind.config.ts
export default {
  theme: {
    extend: {
      colors: {
        primary: {...},
        secondary: {...}
      },
      // Add custom theme options
    }
  }
}

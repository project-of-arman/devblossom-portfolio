1. Introduction

DevBlossom Portfolio is a modern, responsive portfolio template built with React, TypeScript, and Tailwind CSS. It's designed for developers, designers, and creative professionals who want to showcase their work in a beautiful and professional way.

3. Getting Started

3.1 Prerequisites

  • Node.js 16.0 or later
  • npm or yarn package manager

3.2 Installation

Follow these steps to get started:

# Clone the repository
git clone [repository-url]

# Navigate to project directory
cd devblossom-portfolio

# Install dependencies
npm install

# Start development server
npm run dev

4. Customization

4.1 Colors

The color scheme can be customized in the tailwind.config.ts file. The default theme uses CSS variables that can be found in src/index.css.

4.2 Content

To modify the content, edit the following files:

  • src/data/navLinks.ts - Navigation menu items
  • src/data/projectsData.ts - Portfolio projects
  • src/data/servicesData.ts - Services section
  • src/data/skillsData.ts - Skills and expertise

4.3 Components

The template uses a component-based architecture. Main components can be found in:

  • src/components/sections/ - Main section components
  • src/components/common/ - Reusable components
  • src/components/ui/ - UI components from shadcn/ui

5. Deployment

To build for production:

# Build the project
npm run build

# Preview production build
npm run preview

The build output will be in the dist directory, which can be deployed to any static hosting service.
